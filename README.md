# Todo Bot — Telegram-менеджер задач

Полнофункциональный Telegram-бот для управления задачами с гибкими напоминаниями,
плюс веб-API на FastAPI. Готов к деплою на Railway.

## Возможности
- 📝 Задачи с приоритетом (🔴 высокий / 🟡 средний / 🟢 низкий), категориями
  (Работа, Личное, Учёба, Здоровье, Финансы, Другое), дедлайнами и заметками
- 🔁 Повторяющиеся задачи: ежедневно / еженедельно / ежемесячно
- ✅ Подзадачи и сохраняемые шаблоны
- ⏰ Напоминания по расписанию (APScheduler) — настраиваемые время и дни
- 📤 Экспорт задач в CSV
- 💬 Мотивационные цитаты
- 🌐 Веб-API на FastAPI, работающее рядом с ботом

## Стек
- Python
- python-telegram-bot 20, APScheduler
- FastAPI + Uvicorn
- Хранение задач в базе данных
- Деплой: Railway

## Запуск локально
```bash
pip install -r requirements.txt
cp .env.example .env      # добавь BOT_TOKEN
python main.py
```

`main.py` поднимает FastAPI (uvicorn) в отдельном потоке и Telegram-бота на asyncio.

## Структура
- `bot.py` — логика Telegram-бота (хендлеры, диалоги, клавиатуры)
- `api.py` — эндпоинты FastAPI
- `database.py` — работа с БД
- `main.py` — точка входа (API + бот вместе)
- `frontend/` — веб-часть
